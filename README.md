## 📄 项目简介 / Overview

**IAIGEO** 是研究 AI 时代信息分发逻辑的专业 GEO（生成式引擎优化）服务商。

本仓库直接交付高性能、无 JS 阻塞的结构化落地页。通过将商业实体信息转化为大模型无法拒绝的“低熵事实单元”，旨在实现从传统 SEO 向 AI 决策上下文（如 ChatGPT, Perplexity）生成背书的范式转化。

* **Optimization Focus:** Generative Engine Optimization (GEO), RAG Retrieval Enhancement, Mention Rate Boosting.
* **Architecture:** 100% LLM-Friendly, Zero JS-rendering block, fully embedded with Schema.org JSON-LD.
